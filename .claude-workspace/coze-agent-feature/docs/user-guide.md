## 功能介绍

生活物联网智能体管理平台是一个面向儿童智能玩具的智能体管理系统，提供以下功能：

- **智能体创建**: 通过友好的表单创建专属的 AI 助手
- **智能体编辑**: 修改已创建的智能体配置
- **智能体发布**: 将智能体发布到扣子商店
- **实时通话**: 与智能体进行语音对话交互
- **Prompt 生成**: 自动生成儿童友好的 Prompt 模板
- **插件配置**: 选择诗词、天气等技能插件

## 快速开始

### 1. 打开应用

访问应用地址：`/bot-manager`

### 2. 登录

首次使用需要登录：

![登录界面](#)

1. 点击右下角「设置」按钮
2. 输入你的 Coze User ID
3. 输入你的 Personal Access Token (PAT)
4. 点击「确定」保存

> **提示**: PAT 可以在 Coze 开发者平台获取

### 3. 创建智能体

1. 点击首页「创建智能体」按钮
2. 填写智能体信息
3. 点击「创建」完成

## 智能体管理

### 创建智能体

#### 基本信息设置

![创建表单](#)

| 字段 | 说明 | 必填 |
|------|------|------|
| 头像 | 上传智能体头像图片 | 否 |
| 名称 | 智能体的显示名称 | 是 |
| 描述 | 智能体的功能描述 | 是 |

#### Prompt 配置

**自定义描述**: 输入你希望智能体扮演的角色

**回复风格**: 选择 AI 回复的详细程度
- 简洁明了: 简短回复
- 适中详细: 平衡回复
- 丰富详尽: 详细回复

**价值观**: 多选，选择希望 AI 传递的价值观
- 正直、善良、勇敢、真诚等

**习惯培养**: 多选，选择希望培养的好习惯
- 好好吃饭、爱阅读、讲文明等

**技能插件**: 多选，选择智能体拥有的技能
- 今日诗词: 可以朗诵诗词
- 百度天气插件: 可以查询天气
- 亲子关系问题清单: 可以回答亲子问题

#### 语音配置

**音色选择**: 从下拉列表选择 TTS 音色
- 支持搜索音色名称
- 滚动加载更多音色
- 不选择时不使用 TTS

**语速调节**: 调整语音播放速度
- 范围: 0.5x - 2.0x
- 默认: 1.0x

#### 开场白配置

**欢迎语**: 用户进入通话时听到的第一句话

**建议问题**: 设置建议用户问的问题（最多 4 个）
- 点击「添加问题」按钮
- 输入问题内容
- 点击「确定」添加
- 点击 × 删除问题

#### Prompt 预览

点击「预览 Prompt」查看生成的完整 Prompt，包含：
- 字符数统计
- 超出警告（超过 10000 字符）

### 编辑智能体

1. 在智能体卡片上点击「编辑」按钮
2. 修改需要更改的内容
3. 点击「保存」完成

> **注意**: 不上传新头像时，原有头像保持不变

---

### 发布智能体

1. 在智能体卡片上点击「发布」按钮
2. 等待发布完成
3. 发布成功后状态变为「已发布」

## 通话功能

### 进入通话

1. 在智能体卡片上点击「通话」按钮
2. 进入通话页面

### 通话页面说明

![通话页面](#)

#### 空闲状态
- 显示智能体头像和名称
- 显示智能体描述
- 绿色按钮：开始通话

#### 呼叫中
- 显示音浪动画
- 显示"正在连接..."
- 红色按钮：挂断

#### 通话中
- 左侧：智能体信息、通话时长、麦克风控制、挂断按钮
- 右侧：聊天消息区域、音量控制
- 实时显示对话内容

### 高级配置

点击「高级配置」按钮可以设置：

| 配置项 | 说明 |
|--------|------|
| 对话模式 | 连续对话（持续监听）/ 单轮对话（说完即停） |
| 回复模式 | 流式模式（边说边显示）/ 音字同步（说完显示） |
| 静音时长 | 检测到静音多久后结束对话 |

### TTS 设置

1. 点击「TTS 设置」按钮
2. 选择音色
3. 调节语速
4. 点击「保存设置」

> **注意**: TTS 设置重新连接后生效

### 音频配置

1. 点击「音频配置」按钮
2. 选择麦克风设备
3. 调节音量
4. 开启/关闭 AI 降噪
5. 开启/关闭回声消除

## 常见问题

### Q: 忘记保存 PAT 怎么办？

A: 点击右下角「设置」按钮，可以重新输入。如需清除登录信息，点击「清除登录」。

### Q: 创建智能体失败？

A: 检查以下几点：
- PAT 是否正确
- 网络连接是否正常
- 名称和描述是否填写

### Q: 编辑后音色配置丢失？

A: 音色配置保存在本地存储中，确保：
- 在编辑页面选择了音色
- 点击「保存」按钮
- TTS 设置中也保存了配置

### Q: 通话没有声音？

A: 检查以下几点：
- 浏览器是否允许麦克风权限
- 音量是否调至最低
- 麦克风设备是否选择正确
- TTS 是否配置了音色

### Q: 音色列表加载不全？

A: 音色列表支持分页加载：
- 在下拉列表中向下滚动
- 自动加载下一页（每页 50 个）

### Q: Prompt 字符数超限？

A: 优化建议：
- 减少自定义描述内容
- 减少选择的习惯和价值观
- 简化回复风格

### Q: 发布失败？

A: 检查以下几点：
- 智能体是否已填写完整信息
- 网络连接是否正常
- PAT 是否有发布权限

### Q: 知识库功能？

A: 知识库功能正在开发中，敬请期待。

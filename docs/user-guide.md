## 1. 功能概述

知识库管理功能允许您创建和管理知识库，为智能体提供知识支持。主要功能包括：

- 创建文本/图片类型知识库
- 上传本地文件或在线网页
- 管理知识库中的文件
- 将知识库关联到智能体

---

## 2. 快速开始

### 2.1 登录系统

首次使用需要登录 Coze 账号：

1. 点击页面右上角的「登录」按钮
2. 输入您的 Personal Access Token (PAT)
3. 点击「确定」完成登录

### 2.2 创建第一个知识库

1. 在知识库列表页点击「新建知识库」
2. 填写知识库名称（必填）和描述（可选）
3. 上传知识库图标（可选）
4. 选择知识库类型：
   - **文本类型**：支持文档、网页等文本内容
   - **图片类型**：支持图片上传和描述
5. 点击「创建」完成

## 3. 知识库管理

### 3.1 知识库列表

知识库列表页展示所有已创建的知识库，每个知识库卡片显示：

- **知识库名称**和**描述**
- **知识库类型**标签
- **统计数据**：文档数、分段数、使用次数
- **文件大小**和**更新时间**

### 3.2 操作按钮

每个知识库卡片提供以下操作：

| 按钮 | 功能 | 说明 |
|------|------|------|
| **管理** | 进入知识库详情 | 查看和管理文件 |
| **编辑** | 编辑知识库信息 | 修改名称、描述、图标 |
| **删除** | 删除知识库 | 删除知识库及其所有文件 |

### 3.3 搜索和筛选

- **搜索框**：按知识库名称或描述搜索
- **类型筛选**：按知识库类型筛选
- **排序**：按创建时间或更新时间排序

## 4. 文件管理

### 4.1 文本知识库

文本知识库支持以下文件类型：

- **文档**：PDF、TXT、DOC、DOCX
- **网页**：在线网页 URL

#### 上传本地文件

1. 点击「上传文件」按钮
2. 选择「本地文件」标签
3. 点击上传区域或拖拽文件到上传框
4. 等待文件处理完成

#### 添加在线网页

1. 点击「上传文件」按钮
2. 选择「在线网页」标签
3. 输入网页 URL
4. 输入文件名称
5. （可选）配置自动更新设置
6. 点击「添加」

#### 文件操作

| 操作 | 说明 |
|------|------|
| **编辑** | 修改文件名、设置自动更新 |
| **删除** | 删除单个或批量删除文件 |
| **刷新** | 重新加载文件列表 |

### 4.2 图片知识库

图片知识库支持图片上传和描述编辑。

#### 图片标注方式

创建知识库时选择标注方式：

- **系统自动标注**：上传后自动生成图片描述
- **手工标注**：上传后需要手动设置描述

#### 上传图片

1. 点击「上传图片」按钮
2. 选择图片文件（支持 JPG、PNG 等格式）
3. 如果选择手工标注，输入图片描述
4. 等待处理完成

#### 编辑图片描述

1. 找到需要编辑的图片
2. 点击「编辑描述」按钮
3. 修改描述内容
4. 保存更改

### 4.3 文件状态说明

| 状态 | 图标 | 说明 |
|------|------|------|
| **处理中** | 🔄 | 文件正在解析，请稍候 |
| **完成** | ✅ | 文件处理完成，可以使用 |
| **失败** | ❌ | 文件处理失败，请重新上传 |

### 4.4 进度追踪

文件上传后会显示进度弹窗，实时显示：

- 文件名称
- 处理进度（百分比）
- 已耗时（秒）

处理完成后自动关闭弹窗。

## 5. 常见问题

### Q1: 支持哪些文件格式？

**文本知识库**：PDF、TXT、DOC、DOCX、在线网页

**图片知识库**：JPG、PNG、GIF 等常见图片格式

### Q2: 文件大小限制是多少？

- 单个文件最大 10 MB
- 每次最多上传 20 个文件

### Q3: 为什么文件一直显示「处理中」？

文件处理时间取决于文件大小和类型。通常：

- 文本文档：几秒到几分钟
- 大型 PDF：可能需要更长时间
- 图片：通常几秒钟

如果超过 5 分钟仍未完成，建议刷新页面或重新上传。

### Q4: 如何删除知识库？

1. 在知识库列表页找到要删除的知识库
2. 点击「删除」按钮
3. 确认删除操作

**注意**：删除知识库将同时删除其中所有文件，此操作无法撤销。

### Q5: 如何将知识库关联到智能体？

1. 进入智能体创建/编辑页面
2. 找到「知识库选择」部分
3. 勾选需要关联的知识库
4. 保存智能体配置

智能体在对话时将能够检索和使用关联知识库中的内容。

### Q6: 在线网页的自动更新是什么？

自动更新功能可以定期重新抓取网页内容，保持知识库内容最新。

- **不自动更新**：仅在上传时抓取一次
- **自动更新**：按设置的时间间隔定期抓取（最小 24 小时）
